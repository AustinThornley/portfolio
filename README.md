## Run Locally

Clone the project

```bash
  git clone https://github.com/AustinThornley/portfolio.git
```

Go to the project directory

```bash
  cd portfolio/
```

Simply open using a LiveServer extension via your IDE, or simply open the index.html file in any web browser.
